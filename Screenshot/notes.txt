Docker images can be downloaded from hub.docker.com
Docker Desktop to view images in UI

Commands
1.  docker pull nginx (to pull images)
2.  docker run nginx:latest (localhost:80, Welcome to nginx)
3.  docker ps (to view list of running container)
4.  docker ps -a  (to view list of all containers (running and non-running))
5.  docker run -d nginx:latest (to run container in detached mode in background)
6.  docker start containerID (to restart stopped container without creating new containerID)
7.  docker stop container (to stop container)
8.  docker network ls (to list available networks locally)
9.  docker network create network name (to create network)
10.  docker logs (to check logs)
11.  docker run -d -p 27017:27017 (port mapping in detached mode)


Docker project
1.  docker pull mongo
2.  docker pull mongo-express
3.  docker network ls 
4.  docker network create mongo-network
5.  Go to hub.docker.com and copy the code in mongo with variables
6.  docker run -d -p 27017:27017 -e MONGO_INITDB_ROOT_USERNAME=admin -e MONGO_INITDB_ROOT_PASSWORD=12345 --net mongo-network --name mongodb mongo
(to create mongo container with network settings)
7.  docker logs mongodb (to verify container)
8.  docker run -d -p 8081:8081 -e ME_CONFIG_BASICAUTH_USERNAME=jaffar -e ME_CONFIG_BASICAUTH_PASSWORD=jaf123 -e ME_CONFIG_MONGODB_ADMINUSERNAME=admin -e ME_CONFIG_MONGODB_ADMINPASSWORD=12345 -e ME_CONFIG_MONGODB_SERVER=mongodb --net mongo-network --name mongoexpress mongo-express
(create and run mongo-express container with network settings)
9.  docker logs mongoexpress (to verify)
10.  localhost:8081 (login jaffar, jaf123) (mongodb and mongoexpress loaded and connected successfully)
11.  create user (user_db), create collection (users)(collection means tables)

Run and stop app
docker compose up -d
docker compose down

URL
Mongoexpress - http://localhost:8081
FASTAPI - http://localhost:3001/docs
React - http://localhost:3000
